== Description ==

Auto Loan Calculator is a fast and user friendly way for you to present a good and accurate estimate for users needing to get auto loan payment information fast. This widget can be deployed easily and needs no configuration.
If you have any questions or problems, let me know: <a href="http://www.usedcarsgroup.com/" title="Used Cars Group">Used Cars Group</a>

== Installation ==

1. Upload `carloancalc.zip` to the `/wp-content/plugins/` directory and unzip
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget to a valid placement and click save
4. You are done.
